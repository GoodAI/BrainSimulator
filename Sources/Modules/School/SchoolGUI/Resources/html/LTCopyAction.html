LTCopyAction

Name Imitate movement
Implementing class LTCopyAction
Description The agent learns to copy teacher's actions. It observes what effect a teacher's action had on the environment and it tries to achieve similar effect on the environment.
Child equivalent Respond to parent's smile
Child age 5 mths
Prerequisites LTMovingTarget;LT Action w/ Cooldown
Fully observable world yes
Input modalities Visual
Actuators 4 (W,A,S,D), all binary 
Spatial patterns yes  
Temporal patterns yes
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an agent at a default position and a teacher at another position.
Training unit goal state The agent waits for teacher's move and then copies an action that the teacher did.
Reward There should be reward for following the teacher's action. Possibly also for waiting if the teacher is doing nothing.
Level success criterion The pupil has not made a mistake in the last 20 training units.
