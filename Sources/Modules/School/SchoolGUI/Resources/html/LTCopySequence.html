LTCopySequence

Name Imitate sequence
Implementing class LTCopySequence
Description The agent learns to reproduce not only a single action of the teacher, but a sequence of actions of the teacher that is executed while the agent waits.
Child equivalent Repeat after me
Child age 10 months
Prerequisites LTCopyAction,LTMultipleTargetsInSequence
Fully observable world yes
Input modalities Visual
Actuators 4 (W,A,S,D), all binary 
Spatial patterns yes  
Temporal patterns yes
Amount of supervision RL   
Distraction yes
Training unit initial state state There is an agent and teacher is presenting sequence of movements to him..
Training unit goal state The agent waits for a signal from the teacher (change of teacher's texture) and then copies all actions that the teacher did until now.
Reward There should be reward for following the teacher's action. Possibly also for waiting before the teacher gives the signal.
Level success criterion The pupil has not made a mistake in the last 20 training units.
