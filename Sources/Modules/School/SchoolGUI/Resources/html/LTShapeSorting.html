LTShapeSorting

Name Detect objects complementarity
Implementing class LTShapeSorting
Description The agent learns which 2 shapes fit together (are compatible, can be stacked)
Child equivalent Put correct shape in the hole
Child age 1.4 y
Prerequisites LTSimpleShape, LTSimpleColor, LTVisualEquivalence
Fully observable world yes
Input modalities Visual
Actuators 4 (W,A,S,D), all binary
Spatial patterns yes  
Temporal patterns yes
Amount of supervision RL
Distraction yes
Training unit initial state Teacher will present random object and differently shaped holes. 
Training unit goal state Agent has to choose what hole has the same shape as object. The motoric output specifies the hole position
Reward Reward after succesfull shape/hole match
Level success criterion The pupil has not made a mistake in the last 20 training units.
