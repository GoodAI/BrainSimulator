LTSimpleAngle

Name Categorize object angle
Implementing class LTSimpleAngle
Description The agent should say what are sinus and cosinus of angle of the observed object relative to him on a continuous scale
Child equivalent Telling direction
Child age 1 month
Prerequisites LTDetectShape, LTDetectColor
Fully observable world yes
Input modalities Visual
Actuators 2 (mouse X a Y axis standing for sin and cos), both continuous 
Spatial patterns yes   
Temporal patterns no
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an object at some position.
Training unit goal state The agent correctly estimated the object angle on a continuous scale from 0 to 1
Reward Depends on Amount of supervision. There is at least a reward for reaching goal state.
Level success criterion The pupil has not made a mistake in the last 20 training units within a 10% tolerance.
