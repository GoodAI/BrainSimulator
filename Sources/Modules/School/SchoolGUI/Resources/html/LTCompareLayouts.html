LTCompareLayouts

Name Detect similar sets
Implementing class LTCompareLayouts
Description The agents learns to identify whether there similar spatial layouts in visual field
Child equivalent Game "What changed?"or "Find differences"
Child age 10 months
Prerequisites LTDetectShape, LTDetectSimilarity,LTDetectDifference
Fully observable world yes
Input modalities Visual
Actuators 2 (0,1), both binary
Spatial patterns yes  
Temporal patterns no
Amount of supervision RL   
Distraction yes
Training unit initial state There is a set of objects at random position presented at each half of visual field. The laout of halves is either similar or differs .
Training unit goal state Agents will learn to detect obejct shapes and layouts in the visual fielad and detect presence of similar layouts.
Reward The succesfull answer is rewarded
Level success criterion The pupil has not made a mistake in the last 20 training units.
