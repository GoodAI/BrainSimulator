LTPongWithBricks

Name Pong with bricks
Implementing class LTPongWithBricks
Ability name Ability to play pong with bricks
Description The agent is presented the pong game, the expectancy is that the agent completes the game by passing all the required levels
Child equivalent Hit the ball, catch the ball, hit target
Child age 5
Prerequisites LTDetectShape, LTMovingTarget
Fully observable world no
Input modalities Visual
Actuators 2 (A,D), both binary
Spatial patterns yes  
Temporal patterns yes
Amount of supervision RL   
Distraction yes
Training unit initial state Agent is in the arena and the ball is bouncing to the walls. Agent learns to catch the ball by paddle.
Training unit goal state In order to successfully complete a training unit, the agent is expected to pass a level
Stopping condition Agent's cumulative reward goes below lower-limit
Reward 1 if the paddle touches the ball, -1 if the ball goes beyond the paddle in the lower-part of the screen
Level success criterion The pupil has not made a mistake in the last 20 training units.
