LTApproach

Name Reach visual target
Implementing class LTApproach
Description The agent should learn to navigate fast towards a randomly positioned target. Initially, target is close, but as learning progresses, it is placed further and further away.
Child equivalent Reach a toy
Child age 8 months
Prerequisites LTDetectShape, LT1DApproach
Fully observable world yes
Input modalities Visual
Actuators W,A,S,D - 4 binary outputs
Spatial patterns yes   
Temporal patterns yes
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an agent at a default position and a reward at random position.
Training unit goal state The agent is close to the reward.
Reward Depends on Amount of supervision. There is at least a reward for reaching goal state.
Level success criterion The pupil has not made a mistake in the last 20 training units larger than 120% of the optimal path length.
