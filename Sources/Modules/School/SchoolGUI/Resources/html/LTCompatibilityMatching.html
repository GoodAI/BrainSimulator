LTCompatibilityMatching

Name Detect objects complementarity
Implementing class LTCompatibilityMatching
Description The agent learns which 2 shapes fit together (are compatible, can be stacked)
Child equivalent Compatibility of objects - which pairs of objects fit together: shoe + foot, two plates one on another, shape-hole jigsaw, tetris
Child age 1.9 y
Prerequisites Simple Shape PD, Shape/Color PD, Visual equivalence
Fully observable world yes
Input modalities Visual
Actuators 2 (0,1), both binary
Spatial patterns yes  
Temporal patterns no
Amount of supervision RL   
Distraction yes
Training unit initial state A training unit begins with 1 shape on different positions (translation) and 1 compatible place, gradually adding more places and shapes
Training unit goal state Agent observes input and responds with correct binary output (compatible: true / false)
Reward Binary, each step
Level success criterion The pupil has not made a mistake in the last 20 training units.
