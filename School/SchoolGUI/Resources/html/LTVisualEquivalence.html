Implementing class LTVisualEquivalence
Description The agent learns to find equivalence between 2 shapes
Child equivalent The first ability needed for solving shape-hole puzzle - realizing that an object matches a given hole.
Child age 1.5 year
Prerequisites LTDetectShape, LTDetectColor
Fully observable world yes
Input modalities Visual
Actuators 2 (0,1), both binary
Spatial patterns yes    
Temporal patterns no  
Amount of supervision RL     
Distraction yes  
Training unit initial state A training unit begins with 2 shapes on different positions (translation), gradually adding more shapes (2 at a time)  
Training unit goal state Agent observes input and responds with correct binary output (equivalent: true / false)  
Reward Binary, each step  
Level success criterion The pupil has not made a mistake in the last 20 training units.
