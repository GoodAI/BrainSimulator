LT2BackBinaryTask

Name Detect objects position in timeseries
Implementing class LT2BackBinaryTest
Description The agent needs to remember last n frames and respond if present frame is equal to the one n-steps ago. Temporal patterns of length n begin with the same spatial pattern so that the agent can response for certain number of steps. First spatial pattern appears only once in temporal pattern.
Child equivalent (Dual) N-back game
Child age 1.5 y
Prerequisites Simple Shape PD, Shape/Color PD, Visual equivalence
Fully observable world yes
Input modalities Visual
Actuators 2 (0,1), both binary
Spatial patterns yes  
Temporal patterns yes
Amount of supervision RL   
Distraction no
Training unit initial state A training unit begins with one or more iregularly alternating shapes
Training unit goal state Agent observes input (n steps) and responds with correct binary output (match n steps into the history: true / false)
Reward Binary, each step
Level success criterion The pupil has not made a mistake in the last 20 training units.
