LTDetectSimilarity

Name Detect similarity in a set
Implementing class LTDetectSimilarity
Description The agents learns to identify whether there are similar objects in visual field
Child equivalent Find two similar toys on carpet
Child age 7 mths
Prerequisites LTDetectShape, LTDetectDifference
Fully observable world yes
Input modalities Visual
Actuators 2 (0,1), both binary
Spatial patterns yes  
Temporal patterns no
Amount of supervision RL   
Distraction yes
Training unit initial state There is a set of objects at random position presented to the agent. The visual field consists either from different objects or two of them are similar
Training unit goal state Agents will learn to detect obejct shapes in the visual fielad and detect presence of similar objects.
Reward The succesfull answer is rewarded
Level success criterion The pupil has not made a mistake in the last 20 training units.
