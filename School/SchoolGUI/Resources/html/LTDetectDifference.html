LTDetectDifference

Name Detect discrepancy in a set
Implementing class LTDetectDifference
Description The agents learns to identify whether there is different object in visual field
Child equivalent Find special cake in bowl of ordinary ones
Child age 5 months
Prerequisites LTDetectShape
Fully observable world yes
Input modalities Visual
Actuators 2 (0,1), both binary
Spatial patterns yes  
Temporal patterns no
Amount of supervision RL   
Distraction yes
Training unit initial state There is a set of objects at random position presented to the agent. The visual field consists either from same objects or one is different
Training unit goal state Agents will learn to detect obeject shapes in the visual fielad and detect presence of different one.
Reward The succesfull answer is rewarded
Level success criterion The pupil has not made a mistake in the last 20 training units.
