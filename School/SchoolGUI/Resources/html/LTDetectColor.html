LTDetectColor

Name Categorize object color
Implementing class LTDetectColor
Description Discriminate between approximate colors (regions in RGB space)
Child equivalent Show me red toy
Child age 6 months
Prerequisites LTDetectBlackAndWhite
Fully observable world yes
Input modalities Visual
Actuators 2 (A,B), both binary
Spatial patterns yes   
Temporal patterns no
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an empty grid. At some position of the grid, there is a square. The color of the square belongs to one of two possible regions in RGB space (e.g., it's bluish or greenish/yellow)
Training unit goal state The agent sets actuator A to 1 if the square is bluish. Otherwise (greenish/yellow), he sets actuator B to 1.
Reward Reward for reaching goal state. Punishment for any other action.
Level success criterion The pupil has not made a mistake in the last 20 training units.
