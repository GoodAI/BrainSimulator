LTMovingTarget

Name Reach moving target
Implementing class LTMovingTarget
Child equivalent Grab a moving toy
Child age 7 months
Prerequisites LTApproach
Fully observable world yes
Input modalities Visual
Actuators W,A,S,D - 4 binary inputs
Spatial patterns yes   
Temporal patterns yes
Amount of supervision RL
Distraction yes
Training unit initial state A training unit begins with a world displaying an agent at a default position and a reward moving from a random position.
Training unit goal state The agent is close to the reward.
Stopping condition The agent did not learn the ability if after the whole amount of training data is given to him the teacher still has not declared the ability learned.
Reward Depends on Amount of supervision. There is at least a reward for reaching goal state.
Level success criterion The pupil has not made a mistake in the last 20 training units larger than 120% of the optimal path length.
